1. PDF Upload       →    POST /api/ingestion/ingest (multipart form)
2. Load PDF         →    PyPDFLoader from LangChain
3. Split into chunks →   RecursiveCharacterTextSplitter (1000 chars)
4. Embed chunks     →    get_embedding() → OpenAI/Ollama/fallback
5. Store in DB      →    PostgreSQL policy_documents table (pgvector)

Response: {"chunks_created": 45, "status": "indexed", ...}

PHASE 2: DATA RETRIEVAL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Your Screenshot          Our Implementation
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. User Query       →    POST /api/ingestion/retrieve (JSON)
2. Embed Query      →    get_embedding(query)
3. Vector Search    →    PostgreSQL pgvector L2 distance similarity
4. Return top-k     →    Top 6 chunks (customizable)

Response: {"chunks": [...], "count": 6, "timestamp": "..."}

ENDPOINT 1: POST /api/ingestion/ingest
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Request:            file: UploadFile (PDF)
Response (200):     {
                      "filename": "policy.pdf",
                      "document_name": "policy.pdf",
                      "chunks_created": 45,
                      "total_pages": 12,
                      "status": "indexed",
                      "timestamp": "2026-07-10T15:30:45.123456"
                    }
Error Handling:     400 (invalid file), 401 (auth), 403 (permission), 500 (error)
