HOW TO RUN THE BACKEND
================================================================================

1. Start Backend Server
   cd RetailPolicyAssistant
   python -m uvicorn app.main:app --host 127.0.0.1 --port 8000

2. Get Demo Token
   curl http://localhost:8000/token

3. Submit Query
   curl -X POST http://localhost:8000/ask \
     -H "Authorization: Bearer <token>" \
     -H "Content-Type: application/json" \
     -d '{"query": "What is our data retention policy?"}'

4. Expected Response
   {
     "query": "What is our data retention policy?",
     "route": "rag",
     "risk": {"risk_level": "low", "reason": "..."},
     "result": {"result": "Data Retention Policy:\n..."},
     "confidence_score": 0.8,
     "escalate": false,

SYSTEM ARCHITECTURE OVERVIEW
================================================================================

Query Flow:
  Input → Validation → Intent Detection → Routing → Agent Processing → Response

Agent Types:
  • RAG Agent: Retrieves from PDF documents using embeddings
  • SQL Agent: Queries vendor database
